## ✨ Sobre o Projeto

O objetivo deste projeto é apresentar minhas habilidades práticas em desenvolvimento web, com uma interface moderna, responsiva e interativa.

O site foi construído utilizando tecnologias puras da web (HTML, CSS e JavaScript), explorando ao máximo animações, organização de código e boas práticas.

---

## 🧠 Principais Destaques

- 🎯 Design moderno com foco em UI/UX
- 📱 Totalmente responsivo (mobile-first)
- ⚡ Performance otimizada para carregamento rápido
- 🎬 Animações suaves e interativas
- 🧩 Estrutura organizada e escalável

---

## 🛠️ Tecnologias Utilizadas

- HTML5 (semântico)
- CSS3 (Flexbox, Grid, animações)
- JavaScript (DOM, eventos e interatividade)
- Three.js (elementos visuais 3D)
- Font Awesome (ícones)
- Google Fonts (tipografia)
